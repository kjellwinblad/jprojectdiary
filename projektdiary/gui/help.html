What is Project Diary


Project Diary is a program that helps you keep trak of your work with a project.

How to use


Every time you have made changes to your project open your project diary program and write down what you have done and whatever you wish and how much time you have spent on it.

The program keeps track of the total time the project has taken. If you want a report on what you have done it's just to export your project diary to a html file frome the file menu.

The graphical user interface(gui)


The gui is build up by two main components. One is a list of events and the second is an editor for the current selected event.

The event list


You can add and delete events by using the corresponding buttons in the bottom of the list. Select an event by clicking on it. A selected event is set to the editor as the current event.

The event editor


The event editor have feelds for editing a comment of the event, the date of the event and the length of the event. To save after editing a event you can press the save button or select an other event in the event list or add a new event.

The menu


In the file menu you can save current diary, create a new one or open a saved diary. You can also export the current diary to an html file witch is useful if you want to create a report of your work with a project.

In the properties menu you can change the orientation of the components and the look of them.
